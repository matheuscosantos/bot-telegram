2.2 - Função getHtml¶

Para capturar o cdigo HTML do site desejado, foi utilizada a bliblioteca requests, no qual permite abrir uma requisio ao site. Na atividade, usamos a funo uReq() e passamos como parmetro uma URL. Aps isso, foi utilizada a funo read() para capturar todo o HTML.
Sendo assim, a funo getHtml recebe a URL e retorna todo o HTML na varivel pgina.

def getHtml(url):   
    uClient = uReq(url)
    pagina = uClient.read()
    uClient.close()
    return pagina

2.3 - Função getLinks¶

Aps utilizar a funo getHtml e pegar o seu retorno, ele ser passado para a funo getLinks, que recebe todo o HTML e utilizando a biblioteca BeautifulSoup, localiza-se todas as tags < a > com a funo findAll, e acessa todas as funes href com a funo get, armazenando assim todos os links e retornando a varivel do tipo lista chamada links.

In [ ]:
def getLinks(pagina):
    links = []
    linksUnicos = []

    page_soup = soup(pagina,"html.parser")

    for link in page_soup.findAll('a'):    
        links.append(link.get('href')) 
    return links

2.4 - Função filterOnlyText¶

Ao capturar os links e retornar seu contedo, a funo getLinks passada para a filterOnlyText, fazendo com que os elementos sejam conferidos, caso tenha algum que no seja texto ou esteja vazio, no adicionado na lista para retorno.

In [ ]:
def filterOnlyText(links):
    linksUnicos = []
    for link in links:
        if type(link) == str: 
            linksUnicos.append(link)
            
    return linksUnicos

2.5 - Função removeDublicates¶

Para remover os elementos duplicados, foi utilizada a funo removeDublicates, que recebe o retorno do filterOnlyText, aps conferir, todos os links so vlidos, no contendo elementos duplicados e que no sejam texto.

In [ ]:
def removeDuplicates(links):
    uniqueList = []    
    
    for elem in links:
        if elem not in uniqueList:
            uniqueList.append(elem)    
            
    return uniqueList

2.6 - Função locateWord¶

Agora com todas as funes de tratamento e captura do contedo de pginas web prontas, elas sero consumidas pela funo locateWord, que recebe como parmetro uma palavra e um link. Fazendo com que a palavra seja buscada no corpo do link.
Aps receber, usada a funo unidecode para remover todos acentos e a funo lower para deixar todas letras minsculas, para facilitar a posterior comparao.
Ao utilizar as funes citadas anteriormente, elas devero estar com a seguinte estrutura:
removeDuplicates(filterOnlyText(getLinks(getHtml(site))))
Tambm para facilitar a busca da palavra no link, removemos os caracteres '-' e '/'.
A lista de retorno contm apenas os links que possuem a palavra desejada.

In [ ]:
def locateWord(word,site):  
    list = []
    word = unidecode.unidecode(word).lower() #remove os acentos e deixa minsculo 

    linksUnicos = removeDuplicates(filterOnlyText(getLinks(getHtml(site))))

    for link in linksUnicos:
        temp = link.replace("-"," ")    #substitui os "-" por " "
        text = temp.replace("/"," ")    # remove os "/" por " "
        if 'http' in text:
            if word in text:
                list.append(link)
    return list

2.7 - Funcionamento do bot¶

Com isso, chegamos a parte do telepot, framework para Python que permite a comunicao com o Telegram. Ele executado na funo handle, que aps iniciada, fica em looping infinito, para sempre receber e enviar as mensagens para os usurios que se comunicam com o Bot. Cada ciclo de execuo controlada pelo while, que contm o time.sleep(10), determinando o tempo de cada looping.
Uma funo importante a glance, que recebe a mensagem do usurio como parmetro e retorna o contedo, tipo e id do usurio que enviou a mensagem. Como o sistema fica em looping, necessrio criar diferentes camadas de interao, sendo elas: * -1 ) Camada para aguardar o recebimento de uma mensagem, caso a conversa tenha sido finalizada, ao receber a mensagem, a varivel nivel incrementada, permitindo passar para a prxima camada. * 0 ) Camada que envia uma mensagem de boas vindas e pergunta qual o jornal no qual o usurio deseja localizar uma notcia. Aps o envio da mensagem, a varivel nivel incrementada. * 1 ) Camada para receber o jornal digitado pelo usurio, remover os caracteres especiais e deixar o texto em minsculo, caso o jornal escolhido seja localizado na varivel do tipo dictionary, chamada sites, enviada uma mensagem ao usurio, solicitando que ele escreva um tema. Aps isso, a varivel nivel incrementada, caso contrrio, recebe o valor 1, voltando para o nvel de solicitao do site desejado. * 2 ) Camada para receber o tema que o usurio digita, aps digitar, so removidos os acentos, caracteres especiais e as letras so transformadas em minsculas, aps isso, o site e a palavra so enviadas como parmetro para a funo locateWord, que est em uma estrutura de for, que envia todos os links localizados para o usurio. Depois, exibido a quantidade de notcias encontradas e a pergunta se o usurio quer fazer uma nova pesquisa enviada. Por ltimo a varivel nivel incrementada. * 3 ) Camada para receber a resposta da pergunta do nvel 2, caso o usurio queira fazer outra busca, a varivel nivel recebe o valor 1, se no deseja continuar, nivel recebe -1. Fazendo o bot entrar em modo de espera.
